Fart是art平台下利用主动调用实现的脱壳方案。本项目修改自FART项目

## 编译

1. 将diff文件复制到相应的目录

2. 执行`git apply xxx.diff`

3. 执行`source build/envsetup.sh`

4. 执行`make update-api`

5. 执行`make systemimage`
